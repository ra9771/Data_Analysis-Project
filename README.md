# Data_Analysis-Project

Analyze sales data to improve customer experience 
Analyze the data to enhance the customer experience using python and python Libraries NumPy, Seaborn, Pandas, Matplotlib
The Sales Analysis Project is a data-driven Python project aimed at analyzing and visualizing sales data to extract meaningful business insights. The primary goal of the project is to identify trends, discover sales patterns, and assist in strategic decision-making using Python's powerful data analysis libraries.


Tools & Technologies Used:

Python (Core Language)
Pandas for data manipulation
NumPy for numerical operations
Matplotlib & Seaborn for data visualization
Jupyter Notebook for interactive coding and visualization
CSV files as data source


Key Features:

Combined and cleaned multiple monthly sales data files
Extracted useful information such as sales per product, city-wise sales, and best sales month
Identified top-selling products and best time for advertisements
Performed correlation analysis to understand customer purchasing behavior
Built insightful visualizations to support findings and trends


Outcomes:

Generated a clear view of sales performance across months and locations
Helped to suggest the best-performing products and ideal times for marketing campaigns
Developed data storytelling skills through interactive visualization
